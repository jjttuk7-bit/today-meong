## 0. 냉정한 진단 — 왜 지금은 유료가 안 되나

| 관점 | 현실 |
|---|---|
| 재방문 이유 | 계정·기록·습관 루프가 없어 **한 번 써보고 끝**. 유료는 리텐션에서 나옴 |
| 콘텐츠 | 테마 5개 + 절차적 생성. **깊이(프로그램/코스/수면)**가 없음 |
| 차별점 | "AI가 기분 맞춤 공간 생성"은 좋은 wedge지만 **아직 체감이 약함** |
| 결제 | 페이월·구독·무료/유료 경계가 **아예 없음** |
| 단위경제 | 세션마다 AI 이미지/TTS 비용 발생 → **쓸수록 적자** 구조 위험 |

**핵심 포지셔닝 (모든 고도화의 기준축):**
> "오늘의 기분에 맞춰 **매번 새로 생성되는 초개인화 힐링 세션**"
> — Calm/Headspace의 정적 라이브러리와 정반대되는 wedge.

## 현재 구현 스냅샷 (기준점)

- **플로우:** 테마 선택 → 기분 입력(퀵무드+자유텍스트) → 시간 선택 → AI 분석 → 명상 재생
- **테마:** 불멍 / 물멍 / 파도멍 / 구름멍 / 비멍 (5종)
- **AI:** OpenAI `gpt-4o-mini`(기분 분석·파라미터·멘트), `gpt-4o-mini-tts`(나레이션), `gpt-image-1`(배경 이미지) — `lib/openai.ts`
- **비주얼:** Canvas 2D 절차적 생성 + bloom/vignette/film-grain — `src/components/Visualizer.tsx`
- **오디오:** Web Audio 합성음 + 바이노럴 비트 + 솔페지오 주파수 — `src/hooks/useSound.ts`
- **부가:** 호흡 가이드(coherent/sleep/box), 크로모테라피, KO/EN i18n
- **인프라:** Vercel (정적 프론트 + `api/` 서버리스 함수). 계정/DB/분석/결제 **없음**

## 고도화 항목 (11개 영역)

### 1. 콘텐츠 깊이 (Content)
- **현재:** 테마 5개, 세션당 단발성 멘트 5줄
- **문제:** "라이브러리"가 없어 매일 열 이유가 없음
- **고도화:**
  - 프로그램/코스화: "7일 불안 완화", "21일 수면 습관" 등 다회차 여정(진도·완료율)
  - 목적별 세션 타입: 수면유도, 낮잠, 집중(포모도로), 아침 리셋, 3분 긴급 진정
  - 수면 스토리: 잔잔한 서사 + 나레이션 (AI 생성 + 큐레이션 혼합)
  - 데일리 추천 1개(습관 트리거)

### 2. 오디오 (Audio)
- **현재:** Web Audio 합성음 + 바이노럴 + 솔페지오, OpenAI TTS 나레이션
- **문제:** 합성음은 그럴듯하나 **녹음 사운드스케이프 질감**엔 못 미침, TTS는 성우 대비 밋밋
- **고도화:**
  - 고품질 녹음 사운드스케이프(필드 레코딩 루프 레이어링, 라이선스). 합성음은 위에 디테일로
  - 음악 레이어(앰비언트 트랙 라이브러리 / AI 음악 생성)
  - 성우급 보이스: ElevenLabs (이미 provider 추상화 설계됨) — 유료 티어 전용
  - 믹서 UX: 비/천둥/음악 볼륨 개별 조절

### 3. 비주얼 (Visual)
- **현재:** Canvas 2D 절차적 + 포스트프로세싱, AI 배경 이미지
- **문제:** wow는 있으나 모바일 성능/일관성/고급감 한 끗 부족
- **고도화:**
  - WebGL/셰이더 전환(Three.js/PixiJS): 유체·볼류메트릭·실사급, GPU로 모바일 성능↑
  - AI 영상 배경(추후): 이미지 → 짧은 루프 영상(Runway/Pika/Veo)
  - 모바일 최적화: 저사양 감지 시 파티클/해상도 다운스케일, `prefers-reduced-motion` 대응

### 4. 개인화·AI 지능 (핵심 차별점)
- **현재:** 기분 입력 → 파라미터/멘트 1회 생성
- **문제:** 개인화가 세션 1회성이라 "나를 아는 앱" 느낌이 없음
- **고도화:**
  - 무드 트래킹: 세션마다 기분 기록 → 추이 그래프/인사이트
  - 적응형 추천: 과거 데이터로 오늘 세션 자동 추천/자동 세팅
  - 저널링: 세션 후 한 줄 기록 → AI 요약/위로 코멘트(구독 후크)
  - 적응형 세션 길이/강도(시간대·이력 기반)
  - (장기) 바이오피드백: 웨어러블 심박/호흡 연동 실시간 반응

### 5. 리텐션·습관 루프 (현재 0점 → 최우선)
- **현재:** 계정·저장·연속기록 전무
- **문제:** 유료 전환의 전제(재방문)가 성립 안 함
- **고도화:**
  - 계정/인증(Supabase Auth/Clerk), 세션 히스토리 저장
  - 스트릭·목표·뱃지, 주간 리포트
  - 리마인더 알림(웹푸시/PWA)
  - 온보딩: 첫 방문 목적 설문 → 30초 내 "아하 모먼트"

### 6. 플랫폼 (Platform)
- **현재:** 웹(Vercel) 단일
- **문제:** 명상은 모바일·백그라운드 재생·잠금화면이 필수 맥락
- **고도화:**
  - PWA화(설치·오프라인·홈스크린) — 가장 저비용 첫 단계
  - 백그라운드 오디오 + Media Session API(잠금화면 컨트롤)
  - (검증 후) React Native/Expo 네이티브 앱 → 스토어 결제/발견성

### 7. 기술 기반 (Infra)
- **현재:** DB·인증·분석·테스트·비용통제 없음
- **문제:** 유료 서비스 최소 요건 미달(가용성/관측성/보안)
- **고도화:**
  - DB(Postgres/Supabase), 분석(PostHog), 에러 트래킹(Sentry)
  - **비용 통제**: AI 이미지/TTS 서버측 캐시(Vercel Blob/S3) + 레이트리밋 + 사전생성 에셋
  - 테스트/CI, 로깅, 상태 모니터링

### 8. 수익화 (Monetization)
- **현재:** 없음
- **고도화:**
  - Freemium 설계: 무료(테마 2개·5분·기본 보이스·AI배경 주1회) / 프리미엄(전 테마·무제한·ElevenLabs·수면 프로그램·히스토리/인사이트)
  - 결제: Stripe 구독 + 무료 체험(7일). 모바일은 스토어 결제
  - 가격 실험: 월/연 요금, 연간 할인

### 9. 신뢰·전문성 (Trust)
- **현재:** 근거·전문성 표기 없음
- **문제:** 웰니스는 신뢰가 구매 동인
- **고도화:** 전문가 감수·크레딧, 과학적 근거 설명, 후기/평점, 성과 스토리

### 10. 성장 (Growth)
- **고도화:** SEO 랜딩(테마별 페이지), 세션 결과 공유 카드(내 오늘의 멍 이미지), 초대/레퍼럴, ASO

### 11. 법무·컴플라이언스 (Legal)
- **고도화:** 개인정보처리방침·이용약관, 건강 관련 면책 고지, 결제/구독 취소 정책, 데이터 보관·삭제(GDPR/국내법)

## ⚠️ 결정적 리스크 — 단위경제(Unit Economics)

현재 구조는 **세션마다 AI 비용**(gpt-image-1 ~$0.04, TTS, gpt-4o-mini)이 발생. 헤비 유저일수록 적자.
**유료화 전에 반드시:**
1. AI 에셋을 **테마×무드 조합으로 사전 생성 후 재사용**(서버 캐시) → 세션당 AI 호출 0에 수렴
2. 실시간 생성은 **프리미엄 전용/횟수 제한**
3. "생성형"은 마케팅 후크로, "재사용"은 원가 절감으로 분리
